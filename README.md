## Sports test task

#### Run application locally
Using `yarn`:
```
yarn start
```
or using `npm`:
```
npm run start
```

#### Build application for production
Using `yarn`:
```
yarn build
```
or using `npm`:
```
npm run build
```

#### Run tests
Using `yarn`:
```
yarn test
```
or using `npm`:
```
npm run test
```

#### Lint according to Airbnb code style
Using `yarn`:
```
yarn lint
```
or using `npm`:
```
npm run lint
```
